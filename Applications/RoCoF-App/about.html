Metric Dictionary

Display Type Definition / Formula
Frequency (Hz) Raw Direct from frequency Value at each timestamp.
RoCoF (Hz/s) Calculated RoCoF = (f_end - f_start) / Δt, where Δt is seconds between adjacent frequency samples.
Total Generation (MW) Raw Generation value from nearest generation record in time.
Estimated Demand (MW) Raw/Selected Uses ND if present; otherwise uses TSD from nearest demand record.
Fuel Dial MW (per fuel) Raw Direct fuel-specific MW fields from generation data (GAS, COAL, NUCLEAR, etc.).
Interconnector Flow MW Raw Direct flow columns from demand data (IFA_FLOW, BRITNED_FLOW, etc.).
Net Interconnector Flow Calculated NET_INTERCONNECTOR_FLOW = Σ(all individual interconnector flows).
Grid RPM Calculated RPM = (Frequency / 50) × 3000.
Torque Demanded Calculated Torque_demand_Nm = TotalGeneration_MW × 3183.
Torque Actual Calculated RoCoF factor = clamp(1 - RoCoF/0.7, 0.2, 1.8), then Torque_actual = Torque_demand × RoCoF factor.
System Flow Status Calculated ΔMW = TotalGeneration - EstimatedDemand; thresholded labels: exporting, importing, balanced.
Estimated Mechanical Inertia by Fuel Calculated Inertia_fuel_GVA = (FuelMW × H_fuel) / 1000, with H from control inputs.
Available Mechanical Inertia Calculated Inertia_available = Σ(Inertia_fuel_GVA across all fuels).
Inertia Delta Calculated ΔInertia = Inertia_available - ConfiguredSystemGVA.
Frequency/RPM Factor Calculated Factor = Frequency / 50.
Instant Generation (Adjusted) Calculated InstantGeneration = TotalGeneration × (Frequency / 50).
Instant Generation Balance Calculated InstantBalance = InstantGeneration - EstimatedDemand.
Instant Demand Coverage (%) Calculated Coverage = (InstantGeneration / EstimatedDemand) × 100.
Instant Inertia (Adjusted) Calculated InstantInertia = AvailableMechanicalInertia × (RPM / 3000)².
Instant Inertia Delta Calculated InstantInertiaDelta = InstantInertia - ConfiguredSystemGVA.

Notes on Alignment and Fallbacks

  • Generation, demand, and inertia values are matched to each frequency frame using nearest timestamp matching with tolerance.
  • Demand fallback logic uses TSD if ND is unavailable.
  • Non-finite values are sanitized before websocket transmission for JSON safety.
  • Charts retain the most recent 600 points (for example, 10 minutes at 1 frame per second).
  • Table datasets are matched by nearest timestamp and can legitimately show “No data available” if a dataset has no parquet rows for the selected window.
